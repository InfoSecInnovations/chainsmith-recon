Guided Mode

A reference for operators using Guided Mode.

What Guided Mode does

  • Agents proactively share tips, explanations, and next-step suggestions in the chat panel.
  • Domain-specific terms show hover tooltips with definitions.
  • After each scan, you receive a summary with suggested next steps.
  • Adjudication, triage, and verification results include extended plain-language explanations.

What Guided Mode does NOT do

  • It never hides features or restricts capabilities.
  • It does not simplify the interface — Standard Mode is not "advanced," it's just quieter.
  • It is not a skill judgment — experienced operators may enable it on an unfamiliar target or attack surface.

Agents that speak up

Agent When it speaks
Triage After a scan completes, pushes an action plan summary.
Adjudicator When severity is adjusted, explains each rubric factor.
Chainsmith When attack chains are identified, explains step-by-step logic.
Verifier When hallucinations are caught, explains what was wrong.
Coach Welcome message on first enable, scope guidance, explanations.

Proactive messages

Proactive messages appear in the chat panel without you asking. A red dot on the chat icon indicates new messages. Each message includes action buttons you can click to follow up.

  • Dismissable — click × to close without responding.
  • Non-blocking — they appear in the chat stream, not as popups.
  • Auto-suppress — dismiss the same type 3 times in a session and it stops appearing until next session.

Turning Guided Mode off

Click the Guided pill badge in the upper-right corner of any page. Or uncheck the toggle on the Settings page.
